Discover Sénégal — Backend

Date : 21 juillet 2026  |  Contexte : Déploiement production + inter-opérabilité plateformes
Stack : Django 6.0.7 + DRF 3.16 + PostgreSQL 16 + Redis 7 + Celery 5.4 + Django Channels 4.2

1. Inter-opérabilité

Compte unique inter-plateformes : Les plateformes partagent la même base de données et le même secret JWT. Un utilisateur connecté sur une plateforme peut accéder aux autres sans se reconnecter.
EndpointMéthodeRôle
/api/v1/auth/login/POSTLogin email + password — émet access/refresh tokens signés avec le secret partagé.
/api/v1/auth/verify/POSTVérifie un token JWT émis par une autre plateforme. Retourne user_id, email, role.
/api/v1/auth/token/refresh/POSTRafraîchit l'access token.
/api/v1/auth/logout/POSTBlacklist le refresh token.
