5. Guide de Lecture du Diagramme

+int id PK

PK = clé primaire auto-incrémentée

+int user_id FK

FK = clé étrangère vers la table indiquée dans la relation

A "1" *-- "1" B

OneToOne — relation exclusive 1 pour 1 (composition)

+List participants

ManyToMany — table de jointure intermédiaire auto-générée

+str field UK

UK = contrainte unique (unique=True)

+JSON amenities

JSON = JSONField sérialisé en PostgreSQL

A "1" --> "*" B

A possède plusieurs B — sens de la clé étrangère

A "*" ..> "0..1" B

Relation nullable (SET_NULL) ou optionnelle

<<A creer>>

Classe non encore implémentée

<<enumeration>>

Classe représentant un TextChoices Django

on_delete=CASCADE

Suppression en cascade si l'objet parent est supprimé

on_delete=SET_NULL

La FK devient NULL si l'objet référencé est supprimé
